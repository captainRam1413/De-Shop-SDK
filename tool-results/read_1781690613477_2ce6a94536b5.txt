   964‚Üí### Theme System Architecture (3 layers)
   965‚Üí
   966‚Üí1. **SSR default** (`layout.tsx`): `<html data-theme="pro-dark">` ensures every server-rendered HTML uses Pro Dark before any JS runs.
   967‚Üí2. **Inline pre-paint script** (`layout.tsx` `<head>`): `(function(){try{var t=localStorage.getItem('deshop-theme');if(t==='pro-dark'||t==='light'||t==='matrix'||t==='phosphor'||t==='amber'){document.documentElement.setAttribute('data-theme',t);}}catch(e){}})();` ‚ÄĒ runs synchronously before body paint, so users with a saved preference see their theme immediately.
   968‚Üí3. **Runtime** (`useDeShopStore.setTheme` + `TerminalLayout.useEffect`): both call `document.documentElement.setAttribute('data-theme', theme)`. The store also persists to `localStorage`. The `useEffect` is a safety net that re-syncs the attribute if it ever drifts from the store value.
